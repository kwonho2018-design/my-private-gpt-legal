RAYUL 한글숲 개인정보 처리방침

마지막 업데이트: 2026년 6월 24일

1. 기본 원칙

RAYUL 한글숲은 유아가 한글의 소리, 자모, 음절을 놀이로 경험하도록 돕는 iOS 학습 앱입니다. 앱은 광고, 추적, 외부 분석 SDK를 사용하지 않으며 학습 데이터는 기본적으로 사용자의 기기 안에 저장됩니다.

앱은 인터넷 연결 없이 사용할 수 있도록 설계되었고, 자체 서버로 학습 기록이나 프로필 정보를 전송하지 않습니다.

2. 수집하지 않는 정보

앱은 아이의 실명, 주소, 전화번호, 이메일, 정확한 생년월일 같은 개인정보를 요구하거나 수집하지 않습니다. 광고 식별자를 이용한 추적, 타사 광고 네트워크, 데이터 브로커 공유도 하지 않습니다.

3. 기기 안에 저장되는 정보

앱 기능 제공을 위해 별명, 캐릭터, 연령대, 학습 진행 상태, 설정, 보호자가 선택한 일부 학습 기록이 기기 안에 저장될 수 있습니다. 이 정보는 아이에게 이어지는 학습 경험과 보호자 대시보드 표시를 제공하기 위해서만 사용됩니다.

말해 보기 기능에서 마이크 사용을 보호자가 허용한 경우 녹음 파일이 기기 안에 저장될 수 있습니다. 녹음은 외부 서버로 전송되지 않으며 보호자가 언제든 삭제할 수 있습니다.

4. 권한 사용

마이크 권한은 아이가 직접 말하기 연습을 녹음하는 선택 기능에서만 사용됩니다. 권한을 허용하지 않아도 주요 학습 활동은 계속 사용할 수 있습니다.

5. 구매 및 외부 링크

현재 버전은 인앱 상품을 제공하지 않습니다. 보호자 영역에는 구매 복원 버튼이 있을 수 있으나, 어린이 모드에는 결제 UI가 표시되지 않습니다. 구매, 데이터 삭제, 외부 정보 확인 같은 보호자용 기능은 보호자 확인 뒤에 접근하도록 구성되어 있습니다.

6. 데이터 삭제

보호자는 앱 안의 보호자 영역에서 저장된 프로필, 학습 기록, 녹음 데이터를 삭제할 수 있습니다. 앱을 삭제하면 iOS 정책에 따라 기기에 저장된 앱 데이터도 삭제됩니다.

7. 문의

개인정보 처리방침 또는 앱 지원 문의는 지원 페이지를 통해 확인할 수 있습니다.
